# 1. PPA Attack Evaluation

These scripts (comprehensive_attack_evaluation_v2_fedmask.py, comprehensive_attack_evaluation_v2_fedselect.py, comprehensive_attack_evaluation_v2_fedcac.py) evaluate the effectiveness of privacy attacks on the FedMask, FedSelect, and FedCAC PFL framework. It simulates an FL environment with heterogeneous data distributions and measures the ability to infer client data properties from their mask vectors.

## Prerequisites

Install dependencies via pip:
```bash
pip install torch torchvision numpy matplotlib scipy scikit-learn pandas tqdm
```

## Usage

Run the scripts from the command line:

```bash
# For FedCAC
python comprehensive_attack_evaluation_v2_fedcac.py [arguments]

# For FedMask
python comprehensive_attack_evaluation_v2_fedmask.py [arguments]

# For FedSelect
python comprehensive_attack_evaluation_v2_fedselect.py [arguments]
```

### Arguments

The script accepts the following command-line arguments to configure the experiment:

#### Basic Settings
- `--domain`: Dataset domain to use. Choices: `digit` (EMNIST/MNIST), `fashion` (FashionMNIST), `cifar10`. Default: `fashion`.
- `--num_clients`: Number of clients in the simulation. Default: `10`.
- `--local_epochs`: Number of local training epochs. Default: `5`.
- `--batch_size`: Batch size for training. Default: `64`.
- `--lr`: Learning rate. Default: `0.01`.
- `--personalization_rate`: Personalization rate for FedCAC (percentage of parameters to mask). Default: `5.0`.
- `--num_evaluation_trials`: Number of independent attack trials to run. Default: `100`.
- `--template_runs_per_class`: Number of runs to average when generating class templates. Default: `20`.
- `--distribution_type`: Method to generate non-IID data. Choices: `random`, `dirichlet`. Default: `random`.

**Random Distribution Options (only used if `--distribution_type random`):**
- `--total_samples_per_client`: Total number of samples per client. Default: `200`.
- `--min_classes_per_client`: Minimum number of classes a client holds. Default: `3`.
- `--max_classes_per_client`: Maximum number of classes a client holds. Default: `5`.
- `--dominant_class_min_ratio`: Minimum ratio of the dominant class in the client's data. Default: `0.7` (70%).

**Dirichlet Distribution Options (only used if `--distribution_type dirichlet`):**
- `--dirichlet_alpha`: Concentration parameter for Dirichlet distribution. Smaller values mean more heterogeneity. Default: `0.5`.
- `--dirichlet_min_classes`: Minimum number of classes to include in the Dirichlet distribution. Default: `3`.
- `--dirichlet_total_samples`: Total samples per client when using Dirichlet. Default: `200`.

## Example Commands

**1. Basic run with default settings (FashionMNIST):**
```bash
python comprehensive_attack_evaluation_v2_fedcac.py
```

**2. Run on Digit dataset (MNIST/EMNIST) with Dirichlet distribution:**
```bash
python comprehensive_attack_evaluation_v2_fedcac.py --domain digit --distribution_type dirichlet --dirichlet_alpha 0.1
```

**3. Run with custom random distribution settings:**
```bash
python comprehensive_attack_evaluation_v2_fedcac.py --distribution_type random --min_classes_per_client 2 --max_classes_per_client 4 --dominant_class_min_ratio 0.8
```



# 2. Mask Stability Evaluation

This script (`mask_stability_evaluation.py`) evaluates the stability of the masks generated by **FedMask** and **FedSelect** algorithms across different data distributions and datasets. It measures how consistent the generated masks are across different training epochs or conditions.

## Configuration Parameters (`Config` Class)

The configuration is managed by the `Config` class within the script. You can modify the default values in the class definition or override them in the main execution block.

### General Settings
- **`method`**: The algorithm to evaluate.
  - Options: `'fedmask'`, `'fedselect'`
- **`dataset`**: The dataset to use.
  - Options: `'mnist'`, `'emnist'`, `'svhn'`, `'qmnist'`, `'cifar10'`
- **`distribution_type`**: Data distribution strategy representing non-IID scenarios.
  - Options: `'random'`, `'dirichlet'`
- **`num_clients`**: Number of clients simulated (default: `10`).
- **`local_epochs`**: Number of local training epochs (default: `50`).
- **`batch_size`**: Training batch size (default: `64`).
- **`lr`**: Learning rate (default: `0.01`).
- **`personalization_rate`**: Percentage of parameters to personalize/mask (default: `5.0`%).
- **`num_evaluation_trials`**: Number of independent trials to run for the evaluation (default: `100`).
- **`stability_metric`**: Metric used to calculate mask stability.
  - Options: `'cosine'` (Cosine Similarity), `'hamming'` (Hamming Distance based)

### Random Distribution Settings
Used when `distribution_type` is `'random'`.
- **`total_samples_per_client`**: Total samples per client (default: `200`).
- **`min_classes_per_client`**: Minimum number of classes a client holds.
- **`max_classes_per_client`**: Maximum number of classes a client holds.
- **`dominant_class_min_ratio`**: Minimum ratio of the dominant class (default: `0.7`).

### Dirichlet Distribution Settings
Used when `distribution_type` is `'dirichlet'`.
- **`dirichlet_alpha`**: Concentration parameter. Smaller values (e.g., 0.1) imply higher heterogeneity (default: `0.1`).
- **`dirichlet_min_classes`**: Minimum number of classes to ensure per client.
- **`dirichlet_total_samples`**: Total samples per client.

## How to Run

Currently, the script receives parameters by modifying the `if __name__ == '__main__':` block at the bottom of the file (lines 280+), not via command-line arguments.

### 1. Single Experiment Mode
To run a specific configuration:
1.  Open `mask_stability_evaluation.py`.
2.  In the `if __name__ == '__main__':` block, set `run_mode = 'single'`.
3.  Modify the specific configuration lines:
    ```python
    config.method = 'fedmask'       # or 'fedselect'
    config.dataset = 'cifar10'      # or 'mnist', 'emnist', etc.
    config.distribution_type = 'random'
    ```
4.  Run the command below.

### 2. Comprehensive Experiments Mode
To run a batch of experiments covering all methods, datasets (excluding CIFAR-10 by default list), and distributions:
1.  Open `mask_stability_evaluation.py`.
2.  In the `if __name__ == '__main__':` block, set `run_mode = 'comprehensive'`.
3.  Run the command below.

### Execution Command

Execute the script using Python:

```bash
python mask_stability_evaluation.py
```
